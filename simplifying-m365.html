Understanding Microsoft Teams

Teams & Channels 101

Standard Channels

  • Visible to all team members
  • Files stored in main SharePoint
  • Great for general collaboration

Private Channels

  • Only visible to specific members
  • Files stored in separate SharePoint site
  • Deleted channels can be restored within 30 days

Channel vs. Chat

Channels

For structured, ongoing collaboration

Chats

For informal, quick conversations

Meetings & Recordings

  • Recordings automatically saved in OneDrive (personal meetings) or SharePoint (channel meetings)
  • Available indefinitely unless retention policy applied
  • Easy sharing with team members

External Collaboration

How to invite guests:

  1. Go to Team → ••• → Add Member
  2. Enter external email address
  3. Configure guest permissions with IT assistance
Guest access must be enabled by your IT admin

Demystifying SharePoint

Integration with Teams

  • SharePoint stores all files from Teams channels
  • Standard channels share a common SharePoint site
  • Private channels have dedicated SharePoint sites
  • Seamless access from both platforms

File Management & Version Control

Folders

  • Simple organization
  • Intuitive structure
  • Easy to navigate

Metadata

  • Advanced filtering capabilities
  • Ideal for larger libraries
  • Customizable columns

Document Recovery

  • Restore documents via SharePoint's Version History
  • Recover deleted files from Recycle Bin (93-day retention)
  • Compare different versions side by side

Permissions and Access

  • Assign permissions via groups (preferred) or individuals
  • Limit editing by clicking ••• → Manage Access on files/folders
  • External partners require guest access setup by IT
Use permission groups instead of individual assignments for easier management

Navigating OneDrive

Purpose & Security

  • Use OneDrive for personal files or limited sharing
  • Files are private by default unless explicitly shared
  • Secure cloud storage with enterprise-grade protection
  • Access files from anywhere, on any device

Sharing & Collaboration

Co-authoring

  • Simultaneous editing allowed
  • Changes sync automatically
  • See who's working on the document

Secure sharing

  • Control permissions precisely
  • Set expiration dates and passwords
  • Access via Share → Link settings

File Management

  • Recover deleted files via OneDrive Recycle Bin (30-day default)
  • Move files easily via ••• → Move to → Select SharePoint destination
  • Sync files for offline access
  • Automatic versioning of documents

Security & Permissions Made Easy

Managing Permissions

  • Check access via SharePoint file details (Manage Access)
  • Request custom permission groups from IT (e.g., "Finance Managers")
  • When employees leave:
    • IT/admin moves files from user's OneDrive
    • Removes user from all groups
    • Reassigns ownership of shared documents

Sensitivity & Data Protection

Sensitivity Labels

  • Apply via file settings (e.g., Confidential, Internal-only)
  • IT provides label setup and configuration
  • Labels can enforce encryption and restrictions

External Sharing

  • Avoid "Anyone with link" for sensitive data
  • Prefer named users with expirations
  • Set password protection when needed
Always verify sharing settings before sending sensitive documents externally

Daily Workflow: Real-life Examples

Project Setup

  • Create standard channels for different aspects
  • Upload files to the appropriate channels
  • Initiate conversations clearly using @mentions
  • Set up recurring meetings in channel calendar

Sensitive Projects

  • Create private channels for confidential topics
  • Files automatically secure and limited to channel members
  • Use sensitivity labels for additional protection
  • Set expiration dates on shared links

Drafting Documents

  • Draft privately in OneDrive
  • Share with limited collaborators initially
  • Use comments and @mentions for feedback
  • Move to SharePoint when finalized for broader access

Employee Departure

  • Admin moves important files to SharePoint
  • Removes user from Teams/groups immediately
  • Reassigns ownership of critical documents
  • Reviews sharing links created by the user

Example: Budget_Q3_2023_FinalReview.xlsx

Scheduled Reviews

  • Quarterly checks on permissions
  • Review inactive teams/channels
  • Archive completed projects
  • Clean up old versions

Offline & Mobile

  • Use OneDrive Sync for offline files
  • Mobile app for easy scanning and upload
  • Set up notifications for important updates
  • Enable biometric security on mobile

Handling Common Technical Concerns

Known Issues

Syncing Delays

  • Symptoms: Changes not appearing across devices
  • Fix: Manual refresh or sync client restart
  • Prevention: Keep client updated

Channel Deletion

  • Symptoms: Accidentally deleted channel
  • Fix: Restore from Teams admin within 30 days
  • Prevention: Set deletion permissions carefully
